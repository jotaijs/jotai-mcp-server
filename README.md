# Jotai MCP Server

## Installation

```bash
$ npm install -g jotai-mcp-server
```

## Config

```json
{
  "mcpServers": {
    "jotai": {
      "command": "jotai-mcp-server",
      "args": [],
      "env": {
        "GITHUB_PERSONAL_ACCESS_TOKEN": "your_github_token"
      }
    }
  }
}
```
